The Stack

Hardware & Software

01 — storage

NAS Server

TrueNAS Scale op custom hardware. 24TB bruikbare opslag, ZFS RAIDZ2. Geen Dropbox, geen Google Drive — alles lokaal, altijd beschikbaar.

02 — compute

Proxmox Cluster

Twee nodes, tientallen containers en VMs. Jellyfin, Nextcloud, Vaultwarden, Immich — alles in productie, alles gemanaged via Proxmox.

03 — network

Eigen netwerk

Pi-hole voor DNS-filtering, WireGuard voor remote access, Nginx Proxy Manager voor reverse proxy. Volledig in eigen handen.

04 — automation

Home Assistant

Lokale smart home automatisering zonder cloud. Geen Alexa, geen Google Home — alles draait lokaal, met volledige controle.

05 — monitoring

Grafana + Prometheus

Real-time dashboards voor CPU, RAM, temperatuur, netwerk en uptime. Want als het stil is, moet je weten waarom.

06 — backup

3-2-1 Backup

Drie kopieën, twee media, één offsite. Restic + Borgmatic geautomatiseerd. Want één backup is geen backup.
